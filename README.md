# CodeAlpha Frontend Development Internship

This repository contains the projects I completed during my **CodeAlpha Frontend Development Internship**. Each project demonstrates different front-end web development concepts using **HTML, CSS, and JavaScript**.

---

## Projects

### Task 1 – Responsive Calculator

A modern calculator with a clean UI and useful features.

**Features**
- Basic arithmetic operations
- Keyboard support
- Bracket support
- Dark/Light mode
- Theme persistence using Local Storage
- Responsive design

**Live Demo**

https://hemnath19-hub.github.io/codealpha_tasks/CodeAlpha_Calculator/

---

### Task 2 – Responsive Image Gallery

An interactive image gallery with filtering and lightbox functionality.

**Features**
- Category filtering
- Lightbox image viewer
- Previous & Next navigation
- Keyboard shortcuts
- Click outside to close
- Dark/Light mode
- Theme persistence
- Fully responsive layout

**Live Demo**

 https://hemnath19-hub.github.io/codealpha_tasks/CodeAlpha_Imagegallery/
