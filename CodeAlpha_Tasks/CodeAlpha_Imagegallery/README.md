# Responsive Image Gallery

A modern and responsive Image Gallery built using **HTML, CSS, and JavaScript**. The project includes image filtering, a lightbox viewer, keyboard navigation, dark mode, and a fully responsive layout for desktop, tablet, and mobile devices.

---

## Features

- Filter images by category
- Lightbox image viewer
- Previous & Next navigation
- Keyboard shortcuts
  - `←` Previous image
  - `→` Next image
  - `Esc` Close lightbox
- Click outside the image to close the lightbox
- Dark Mode / Light Mode
- Theme preference saved using Local Storage
- Fully responsive design
- Smooth hover animations and transitions

## Features Breakdown

### Image Filtering
Display images based on selected categories using JavaScript.

### Lightbox
Click any image to view it in a fullscreen lightbox.

### Navigation
Navigate through filtered images using:
- Previous / Next buttons
- Left & Right Arrow keys

### Dark Mode
Toggle between light and dark themes with a single click.

The selected theme is automatically saved using **Local Storage** and restored when the page is reopened.

### Responsive Design
Optimized layouts for:

- Desktop
- Mobile
- Tablet
